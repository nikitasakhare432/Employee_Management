# 🧑‍💼 Employee Management System

## 📌 Project Overview

The Employee Management System is a backend application developed using **Java Spring Boot**. It provides RESTful APIs to manage employee records such as adding, updating, deleting, and retrieving employee details.

---

## 🚀 Tech Stack

* **Backend:** Java, Spring Boot
* **Database:** MySQL
* **ORM:** Hibernate / JPA
* **API Testing:** Postman
* **Build Tool:** Maven

---

## ✨ Features

* Add new employee
* View all employees
* Update employee details
* Delete employee
* REST API-based architecture
* Exception handling and validation

---

## 📂 Project Structure

src/main/java → Controller, Service, Repository, Model
src/main/resources → application.properties

---

## ⚙️ Setup & Run

1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Configure database in `application.properties`
4. Run the Spring Boot application

---

## 🔗 API Endpoints (Sample)

* `POST /employee/add` → Add employee
* `GET /employee/list` → Get all employees
* `PUT /employee/update/{id}` → Update employee
* `DELETE /employee/delete/{id}` → Delete employee

---

## 📸 API Testing

Use **Postman** to test all endpoints.
